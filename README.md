DESCRIPCIÓN:
-
El proyecto consiste en el sitio web de un gimnasio, DeloxFit, en el que es posible obtener toda la información posible acerca del gimnasio, visualizar y reservar clases guiadas, y ver los diferentes planes de suscripción, existiendo la posibilidad de escoger cualquiera de ellos.

REQUISITOS:
-
REQ1- El sistema debe mostrar la dirección exacta, número de teléfono y correo electrónico de
contacto.

REQ2- El sistema debe mostrar una galería de fotos de las instalaciones disponibles.

REQ3- El sistema ofrece un calendario con las diferentes clases y sus horarios, permitiendo
reservarlas.

REQ4- Se ofrece una relación de los planes de suscripción disponibles y sus características
económicas.

REQ5- Se permite el registro de nuevos usuarios y la autenticación de cuentas existentes
mediante un sistema de inicio de sesión.

REQ6- En caso de credenciales inválidas en el inicio de sesión, el sistema debe mostrar un
mensaje de error: "Usuario o contraseña incorrectos".

REQ7- El sistema debe presentar un formulario de registro que valide que el correo electrónico
no esté previamente registrado.

REQ8- Se requiere obligatoriamente que el usuario esté autenticado en el sistema para
proceder con cualquier transacción económica.
